Wow, I have finished deployed the Hexo for the first time. I have thought to deploy my personal blog on the Tencent Cloud, but it costs some time to buy and deploy a new domain name. To make it worse, it needs identification. So, I give it up. To deploy the blog on github is faster.

How to deploy a Hexo on github

  • Download Nodejs and Npm from the official website
  • Add it to your environment variable (me on linux)
    Open /etc/environment, input following line: /your/nodejs/path after the PATH variable.
  • Install Hexo
  • Initialize hexo For linux users, you could ignore the warings about the fsevents, fsevens is for the mac system.
  • Conifg github
    • Create a repository on github, with then name username.github.io, there can only be one repository with surfix “io”.
    • Config your ssh key for your github, if you don’t have one/
  • Install git addon for your hexo
  • Config your _conifg.yml under your root directory
  • One last step to go! Bingo! you have your first blog now, just open your github blog url.
